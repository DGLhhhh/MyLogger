# MyLogger
我写了一个基于C++的高性能日志库。实现功能：（1）异步日志：使用独立的写入线程，减少对主线程的影响  （2）日志级别：支持 DEBUG, INFO, WARN, ERROR, FATAL  （3）日志文件滚动：可设定单个日志文件最大存储量 （4）多线程安全：使用互斥锁保护共享资源  （5）高性能：异步写入，批量处理  （6）自动清理：自动删除旧日志文件  （7）丰富的日志信息：包含时间戳、级别、文件名、行号、函数名
